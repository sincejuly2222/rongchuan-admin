# 2.原型生成页面
请根据我提供的原型图，生成对应的 React + TypeScript + Ant Design 页面代码。

要求：
1. 严格参考原型图的页面结构
2. 使用 React + TypeScript
3. 优先使用 Ant Design 官方组件
4. 先使用 mock 数据，不要接真实接口
5. 页面代码要可运行、可维护、方便接入现有项目
6. 不要把所有代码都堆在一个超大组件里
7. 样式保持简洁，避免大量混乱的内联样式
8. 如果是后台管理系统风格，请优先按后台页面习惯实现

推荐映射方式：
- 筛选区：Form / Input / Select / DatePicker / Button
- 列表区：Table
- 统计区：Card / Statistic
- 弹窗区：Modal
- 侧边编辑区：Drawer
- 状态展示：Tag / Badge

输出要求：
1. 先简要说明实现思路
2. 给出推荐文件结构
3. 输出主页面代码
4. 如有必要，输出拆分后的子组件代码
5. 给出 mock 数据示例
6. 说明如何接入现有项目
